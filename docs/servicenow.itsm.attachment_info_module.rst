*******************************
servicenow.itsm.attachment_info
*******************************

**a module that users can use to download attachment using sys_id**


Version added: 2.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Download attachment using attachment's sys_id.

Notes
-----

Examples
--------

.. code-block:: yaml

    - name: ServiceNow download attachment module
        servicenow.itsm.attachment:
          instance:
            host: https://instance_id.service-now.com
            username: user
            password: pass
          dest: /tmp/sn-attachment
          sys_id: 003a3ef24ff1120031577d2ca310c74b
